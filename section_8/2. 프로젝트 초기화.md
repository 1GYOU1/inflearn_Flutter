`Tip1`. 클래스 템플릿 작성
- `stless` - 클래스 작성 템플릿
- 하단과 같은 템플릿이 자동 완성 된다.

```dart
class  extends StatelessWidget {
  const ({super.key});

  @override
  Widget build(BuildContext context) {
    return const Placeholder();
  }
}
```
